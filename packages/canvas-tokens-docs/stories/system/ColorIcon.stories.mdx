# System Icon Color Tokens

System icon color tokens provide values for icons.

## Usage

```ts
// styles.ts
import {system} from '@workday/canvas-tokens-web';

const styles = {
  color: `var(${system.color.icon.default})`,
};
```

```css
// styles.css
.icon {
  color: var(--cnvs-sys-color-icon-default);
}
```
