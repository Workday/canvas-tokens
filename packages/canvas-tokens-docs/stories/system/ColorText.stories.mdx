# System Text Color Tokens

System text color tokens provide values for typography.

## Usage

```ts
// styles.ts
import {system} from '@workday/canvas-tokens-web';

const styles = {
  color: `var(${system.color.text.default})`,
};
```

```css
// styles.css
.card-text {
  color: var(--cnvs-sys-color-text-default);
}
```
