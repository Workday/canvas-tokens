# System Size Tokens



## Usage

Size tokens can be consumed as a JavaScript / TypeScript object. Each token value references the
CSS variable from a base token, so you'll need to have those CSS variables imported in your
application as well.

```ts
// styles.ts
import {system} from '@workday/canvas-tokens-web';

const styles = {
  width: `var(${system.size.sm})`,
  height: `var(${system.size.md})`,
};
```

They can also be consumed as CSS, Sass, or Less variables. In the example below, we're applying the
same styles using CSS variables directly.

```css
// styles.css
.button {
  width: var(--cnvs-sys-size-sm);
  height: var(--cnvs-sys-size-md);
}
```

---

## Tokens

These tokens use a multiplier scale and reference our `base.baseline` token, which is `0.5rem` / `8px`.
So the `system.size.sm` token has a value calculated from the baseline multiplier.
