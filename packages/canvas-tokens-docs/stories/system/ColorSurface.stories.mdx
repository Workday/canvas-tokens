# System Surface Color Tokens

System surface color tokens provide values for surfaces.

## Usage

```ts
// styles.ts
import {system} from '@workday/canvas-tokens-web';

const styles = {
  backgroundColor: `var(${system.color.surface.default})`,
};
```

```css
// styles.css
.card {
  background-color: var(--cnvs-sys-color-bg-default);
}
```
