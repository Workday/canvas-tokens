# Border Colors

Border colors are used to define boundaries against a background, or to separate content areas.

## Usage

```ts
// styles.ts
import {system} from '@workday/canvas-tokens-web';

const styles = {
  borderColor: `var(${system.color.border.container})`,
};
```

```css
// styles.css
.card {
  border-color: var(--cnvs-sys-color-border-container);
}
```
